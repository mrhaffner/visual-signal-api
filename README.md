## Visual Signal

Real time project management software - a Trello Clone!

## Features
- User creation / authentication / authorization
- Sharing boards between users
- Real time updates with websockets

## Built with:
* Node / Express
* Typescript
* MongoDB / Mongoose
* Apollo Server / GraphQL
* JSON Web Token / BCrypt
