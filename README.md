### This is the API for my visual signal project - a Trello clone!

#### Main technologies and libraries used:
* Node / Express
* Typescript
* MongoDB / Mongoose
* Apollo Server / GraphQL
* Websockets for real time updates across users
* JSON Web Token / BCrypt
